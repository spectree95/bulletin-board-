# Bulletin Board Marketplace

A full-featured online marketplace built with **Django**, where users can create listings, upload product images, communicate in real time, and manage their own advertisements.

---

# 🌍 Live Demo

**https://bulletin-board-ftyh.onrender.com**

---

# ✨ Features

* 👤 User registration & authentication
* 🔐 Secure login system
* 📦 Create, edit and delete product listings
* 🗂 Categories & subcategories
* 🏷 Dynamic product attributes
* 🖼 Multiple image upload
* ❤️ Like system
* 🔍 Product search & filtering
* 💬 Real-time private chat (Django Channels + WebSocket)
* 📱 Responsive interface
* 🛠 Django Admin Panel

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/spectree95/YOUR_REPOSITORY.git
```

Move into the project

```bash
cd YOUR_REPOSITORY
```

Create virtual environment

```bash
python -m venv venv
```

Activate environment

Windows

```bash
venv\Scripts\activate
```

Linux / macOS

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Apply migrations

```bash
python manage.py migrate
```

Create superuser

```bash
python manage.py createsuperuser
```

Run the server

```bash
python manage.py runserver
```
